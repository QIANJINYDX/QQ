# DD

项目名称：DD

使用环境：

|             环境              |  版本  |
| :---------------------------: | :----: |
| Android Gradle Plugin Version | 7.3.0  |
|        Gradle Version         |  7.4   |
|      Compile Sdk Version      |   32   |
|          Gradle JDK           |   11   |
|            Python             | 3.7.12 |
|             Flask             | 2.2.2  |
|             MySql             | 5.5.62 |

#### 开发目的

主要帮助自我Android开发学习，巩固书本知识,开发一个类似QQ的软件（小QQ）。
